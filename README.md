# SCPU-16 — 一个简单的 16 位 CPU 模拟器与汇编器

## 1. 这是什么？

SCPU-16 是一个用 **C++** 编写的教学型项目，包含：

- 一个 **16 位 CPU 模拟器**，拥有 8 个通用寄存器、64 KB 内存和一套自定义指令集。
- 一个 **汇编器**，能将类汇编源代码转换为可在该模拟器上运行的机器码。
- 从最底层的 **NAND 门** 开始，逐级搭建出逻辑门、全加器、ALU，最后构建出完整的 CPU。

非常适合用来学习计算机组成原理、指令集模拟以及简单的编译器/汇编器前端设计。

## 2. 源代码包括什么？

| 文件 / 模块          | 说明                                                                 |
|----------------------|----------------------------------------------------------------------|
| 主程序文件            | 包含所有核心代码（CPU 定义、ALU、汇编器、测试用例）。                |
| `NAND` → 基础门电路  | 从与非门开始，构建 `NOT`、`AND`、`OR`、`XOR`。                       |
| 一位全加器            | `full_adder(x, y, cin)`，输出 `sum` 和 `cout`。                      |
| 16 位运算器           | `add16`、`sub16`、`and16`、`or16`、`xor16` 等按位运算。              |
| ALU（算术逻辑单元）   | 根据操作码执行 `ADD`、`SUB`、`AND`、`OR`、`XOR`，并输出标志位。      |
| CPU 结构体            | 寄存器组 `reg[8]`、程序计数器 `pc`、标志位 `flags[3]`、内存 `mem[64KB]`。 |
| 指令取指与执行        | `fetch_instruction` 和 `execute_instruction` 实现完整的指令周期。     |
| 汇编器前端            | 词法分析器 `next()`，语法解析与机器码生成 `assemble()`。              |
| 测试用例              | 内嵌 `main()` 中的示例汇编程序，演示加法、存储、停机等操作。         |

## 3. 指令集说明

整个指令集为 **定长 5 字节**，支持 **寄存器寻址** 和 **立即数寻址** 两种模式。

### 指令格式（机器码）
字节0: 操作码 (opcode)
字节1: 目的寄存器编号 (dst_reg, 低4位有效)
字节2: 源操作数类型 (0=寄存器, 1=立即数)
字节3: 若为寄存器模式: 源寄存器编号 (src_reg, 低4位)
若为立即数模式: 立即数低8位 (imm_low)
字节4: 寄存器模式: 固定为 0x00
立即数模式: 立即数高8位 (imm_high)
### 完整指令表

| 助记符 | 操作码 | 说明                                          | 示例                  |
|--------|--------|-----------------------------------------------|-----------------------|
| `MOV`  | `0x01` | Rd = Rs  或  Rd = imm16                       | `MOV R0, R1` / `MOV R0, $5` |
| `ADD`  | `0x02` | Rd = Rd + Rs/imm，更新 C/Z/N 标志              | `ADD R0, R1`          |
| `SUB`  | `0x03` | Rd = Rd - Rs/imm，更新 C/Z/N 标志              | `SUB R0, $1`          |
| `AND`  | `0x04` | Rd = Rd & Rs/imm，更新 Z/N 标志               | `AND R0, R1`          |
| `OR`   | `0x05` | Rd = Rd \| Rs/imm，更新 Z/N 标志              | `OR R0, R2`           |
| `XOR`  | `0x06` | Rd = Rd ^ Rs/imm，更新 Z/N 标志               | `XOR R0, R3`          |
| `CMP`  | `0x10` | 计算 Rd - Rs/imm，仅更新标志（不修改 Rd）     | `CMP R0, $10`         |
| `JMP`  | `0x20` | 无条件跳转到 imm16 （忽略 Rd 字段）           | `JMP $0x100`          |
| `JZ`   | `0x21` | 若 Zero 标志=1，跳转到 imm16                  | `JZ $end`             |
| `JNZ`  | `0x22` | 若 Zero 标志=0，跳转到 imm16                  | `JNZ $loop`           |
| `LOAD` | `0x30` | Rd = MEM[Rs/imm]                             | `LOAD R0, $0x1000`    |
| `STORE`| `0x31` | MEM[Rs/imm] = Rd                             | `STORE R0, $0x1000`   |
| `HALT` | `0xFF` | 停机（程序结束）                              | `HALT`                |

> 标志位：**C**（进位/借位）、**Z**（结果为零）、**N**（结果为负）。  
> 立即数前缀为 `$`，支持十进制（`$10`）和十六进制（`$0x1A`）。  
> 寄存器为 `R0`～`R7`（不区分大小写）。
